HTML dosyası nedir?

HTML'in açılımı Hyper Text Markup Language (çok yere açılan metin).
Bir HTML dosyası bir text dosyası olmakla beraber markup tags (işaretlenmiş etiketler) içerir.
Markup tag'ler web tarayıcınıza sayfanın nasıl gösterileceğini söyler.
Bir HTML dosyasının uzatntısı htm ya da html olmak zorundadır.
Bir HTML dosyası basit bir text editörü ile oluşturulabilir.
